But it's important to know that we can only add the same type of values. Numbers with numbers and Strings with Strings. To add different types of values we need to convert it depending on what we are trying to do. For example:

age = 18
print("I'm " + str(age) + " years old") -----> result: I'm 18 years old

number = "19" --- A number inside quotation marks is considered a String
print(int(number) + 45) -----> result: 64

We have three different functions to convert values:
str() to convert numbers to Strings ------> str(18) --- result: "18"
int() to convert Strings or floating numbers to integer numbers ------> int("18") --- result: 18
float() to convert Strings or integer numbers to floating numbers ------>float("18") --- result: 18.0

You are prepared to learn how to use if-else conditional statements. But first, you must learn the rules that are mandatory for every if-else conditions:

  1. Your condition always needs to have at least one conditional comparator. Example: greater than (>), less than (<), greater or equal to (>=), less or equal to(<=), equal (==).
  2. You must compare the same type of values: Example: "String" == "String" or number <= number
  3. At the end of every if-else you must use a colon (:)
  4. To set the instructions if the condition is true or if not, you must leave a blank space on each line.
