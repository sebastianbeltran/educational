5. It's time to create a variable to store information.
Create a variable with the name language and store the String "Python" in it. Use the print() function to write the message already stored inside the variable.

Perfect! Remeber you can store information as words or numbers inside variables to create messages or math operations. For example:

language = "Python"
print(language + "is the programming language that I'm using")

number = 1
print(number + 22)

But it's important to know that we can only add the same type of values. Numbers with numbers and Strings with Strings. To add different types of values we need to convert it depending on what we are trying to do. For example:

age = 18
print("I'm " + str(age) + " years old") -----> result: I'm 18 years old

number = "19" --- number inside quotation marks
print(int(number) + 45) -----> result: 64

We have three different functions to convert values:
str() to convert numbers to Strings ------> str(18) --- result: "18"
int() to convert Strings or floating numbers to integer numbers ------> int("18") --- result: 18
float() to convert Strings or integer numbers to floating numbers ------>float("18") --- result: 18.0
